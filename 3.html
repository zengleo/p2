Interactive Dataset

The dataset is used module in the next page. This module is useful for comparing the test scores of different values of a field. Below is a short tutorial on how to use this module.

How to Use

Step 1: Select the field you would like to compare.

Step 2: Select the value of the field you would like to graph

Step 3: Hit the "Refresh" Button

It is best to limit comparisons to values of the same field, in order to avoid overlapping of data.

The graph will plot the pretest scores on the x-axis and posttest scores on the y-axis, for the field and value selected. The diagonal line is where pretest scores = posttest scores. Data points towards the upper left represent students with the most score improvement, and data points towards the upper right represent students with the higher scores.
